# Automatizar-inicio-de-CMD
Automatiza los siguiente: inicio de ventanas a mostrar con sus colores cada uno, aunque también se le puede añadir un colorScheme.

Para instalar el colorScheme personalizado se debe:
1. presionar `windows + ,` en la CMD
2. Click en "Abrir archivo JSON"
3. Ubicar "schemes" y pegar en los corchetes
